# Anniversary Music Setup Instructions

To add background music to your anniversary page:

1. Add your music file to the `music/` folder
2. Name it `anniversary-song.mp3` (or `anniversary-song.ogg` for better browser compatibility)
3. Supported formats: MP3, OGG, WAV

## Recommended Music:
- Romantic instrumental pieces
- Soft piano or acoustic guitar
- Love-themed songs (instrumental versions preferred)
- Keep file size under 5MB for fast loading

## File Structure:
```
music/
  anniversary-song.mp3
  anniversary-song.ogg (optional)
```

The music will automatically start playing (looped) when visitors successfully enter the password and unlock the anniversary page.

## Browser Compatibility:
- Modern browsers support MP3
- OGG provides fallback for some browsers
- Music respects browser autoplay policies
